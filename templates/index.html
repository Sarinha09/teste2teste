A Inteligência por Trás do AstroBit

Nosso modelo foi treinado com milhares de pontos de dados do Telescópio Espacial Kepler para classificar candidatos a exoplanetas com alta precisão e confiabilidade.

Random Forest Classifier

Optamos pelo Random Forest por sua robustez, alta performance em dados complexos e, crucialmente, pela sua capacidade de interpretar quais características (features) são mais importantes para a classificação, algo essencial para a validação científica.

Fluxo de Trabalho do Modelo

  1. 1
    Coleta e Preparação de Dados

    Utilizamos o dataset público do Kepler, tratando dados ausentes e normalizando as features para o treinamento.

  2. 2
    Treinamento e Validação

    O modelo foi treinado com 70% dos dados e validado com os 30% restantes, usando a técnica de Validação Cruzada (Cross-Validation).

  3. 3
    Implantação

    O modelo finalizado foi integrado à plataforma para fornecer classificações em tempo real.

Métricas de Performance Reais

  • Acurácia: ...%

    Percentual de previsões corretas do modelo.

  • Precisão (Média): ...%

    Dos "planetas" previstos, quantos eram realmente planetas.

  • Recall (Média): ...%

    De todos os planetas reais, quantos o modelo encontrou.

Matriz de Confusão
